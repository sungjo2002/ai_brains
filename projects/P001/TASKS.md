# TASKS

## 현재 확인 필요

1. PC2에서 동기화 후 아직 늦게 따라오는 화면이 있는지 추가 확인한다.
2. 차량 담당자/기본 운전자 선택 목록이 사업장/근로자 변경 후 바로 갱신되는지 확인한다.
3. PC ↔ PC 서버 백업/복구 동기화 테스트를 진행한다.
4. 근로자 얼굴 사진 동기화 실제 구현 여부를 확인한다.
5. 사업장 이동 시 해당 사업장 근로자의 사업자 값 자동 보정이 필요한지 코드 기준으로 확인한다.

## 우선 테스트

### PC_A / PC_B 한 컴퓨터 테스트

- 한 PC에서 두 프로그램을 테스트할 때는 폴더를 나눈다.
  - 예: `C:\green_test\PC_A`
  - 예: `C:\green_test\PC_B`
- 서버 주소는 같게 둔다.
- 로컬 데이터/DB는 달라야 한다.
- 백업 위치는 가능하면 다르게 둔다.
- PC_A에서 동기화 전 등록한 자료가 PC_B에 바로 보이면 같은 로컬 DB를 쓰는 상태라 테스트가 정확하지 않다.
