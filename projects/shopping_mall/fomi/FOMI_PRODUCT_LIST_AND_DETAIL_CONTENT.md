# Fomi LED 상품명 및 상세페이지 내용 정리

이 문서는 포미 LED 조명 상품을 쇼핑몰에 등록하거나 상세페이지를 만들 때 바로 보기 좋게 정리한 기준 문서다.

## 1. 작업 기본 원칙

- 가격 문구는 대표이미지와 상세페이지에 넣지 않는다.
- 배송 안내는 넣지 않는다.
- 교환 안내는 필요 시 별도 섹션으로만 넣는다.
- 대표이미지에는 글자, 가격, 배송 문구, 로고, 워터마크를 넣지 않는다.
- 상품 이미지는 사용자가 준 실제 상품 사진의 외형을 기준으로 한다.
- 상품별 모듈 개수와 리모컨 유무를 틀리게 만들지 않는다.
- 파일명은 한글 없이 영어와 언더바 `_`로 저장한다.

## 4. 상품별 상세페이지 핵심 내용

### 4-1. 포미 LED 거실등 150W 삼색변환 밝기조절 리모컨

**상품 포지션**

- 넓은 거실용 메인 LED 거실등
- 삼색변환과 밝기조절을 리모컨으로 조작하는 상품
- 고급스럽고 깔끔한 사각 라운드 디자인 강조

**대표 문구**

```text
공간 분위기에 맞춰 조명색을 선택하는 LED 거실등
리모컨으로 색상과 밝기를 간편하게 조절
넓은 거실을 밝고 깔끔하게 연출하는 150W 조명
```

**상세페이지 구성 포인트**

- 전구색 / 주백색 / 주광색 3가지 색상 표현
- 밝기조절 기능 표현
- 리모컨 조작 이미지 또는 리모컨 설명
- 거실 메인등으로 적합한 이미지
- 제품 스펙표 포함

**스펙 예시**

| 항목 | 내용 |
|---|---|
| 제품명 | 포미 LED 거실등 150W 삼색변환 밝기조절 리모컨 |
| 소비전력 | 150W |
| 정격 | 220V / 60Hz / 150W |
| 안전인증 | HD11260-23032 |
| 제조원 | ㈜에이아이조명 |
| 사이즈 | W800 × D800 × H45mm |
| 색상 기능 | 전구색 / 주백색 / 주광색 |
| 조작 방식 | 리모컨 |
| 설치 방식 | 천장 설치형 |

주의: 사용자가 요청한 경우 제조국 / MADE IN KOREA 문구는 넣지 않는다.

### 4-6. 포미 LED 거실등 180W 국산 친환경조명

**상품 포지션**

- 넓은 거실용 고출력 LED 거실등
- 긴 라운드 사각 모듈 3개 구성
- 큰 공간을 밝게 연출하는 상품

**대표 문구**

```text
넓은 거실을 환하게 밝혀주는 180W LED 거실등
3개 모듈 구성으로 공간 전체를 균형 있게 연출

```

**외형 주의사항**

- 반드시 3개 모듈로 표현한다.
- 1개 또는 2개 모듈로 만들지 않는다.
- 모듈은 긴 라운드 직사각형 형태를 유지한다.

**상세페이지 구성 포인트**

- 넓은 거실 설치 컷
- 3개 모듈 강조
- 밝고 균일한 조명감
- 제품 스펙표

## 5. 상세페이지 공통 구성 템플릿

### 5-1. 상단 메인 영역

```text
[상품명]
공간을 밝고 깔끔하게 완성하는 LED 조명
심플한 디자인 / 안정적인 밝기 / 천장 밀착형 설치
```

이미지:

- 제품 대표 이미지
- 실내 설치 컷
- 배경은 제품 조명 색상과 맞춤

### 5-2. 핵심 장점 영역

```text
POINT 01. 깔끔한 천장 밀착형 디자인
POINT 02. 공간을 밝게 채우는 LED 조명
POINT 03. 다양한 실내 공간에 어울리는 심플한 외형
```

리모컨형 상품은 아래 포인트 추가:

```text
POINT 04. 리모컨으로 간편하게 조작
POINT 05. 전구색·주백색·주광색 삼색변환
POINT 06. 밝기조절로 상황에 맞게 사용
```

### 5-3. 디자인 설명 영역

```text
부드러운 라운드 모서리
얇고 깔끔한 테두리 라인
천장에 자연스럽게 밀착되는 슬림한 구조
실내 분위기를 해치지 않는 심플한 화이트 디자인
```

### 5-4. 공간별 추천 영역

| 상품 종류 | 추천 공간 |
|---|---|
| 거실등 | 거실, 넓은 방, 가족 공간 |
| 방등 | 침실, 아이방, 공부방, 작은 방 |
| 주방등 | 주방, 조리 공간, 다용도실 |

### 5-5. 스펙표 영역

기본 스펙표는 아래 항목을 사용한다.

| 항목 | 내용 |
|---|---|
| 제품명 | 상품명 그대로 입력 |
| 소비전력 | 상품명 기준 W 입력 |
| 조명 방식 | LED |
| 설치 방식 | 천장 설치형 |
| 디자인 | 라운드 / 사각 / 원형 / 슬림형 등 상품별 입력 |
| 색상 기능 | 리모컨형은 삼색변환, 일반형은 확인된 색상만 입력 |
| 조작 방식 | 리모컨형은 리모컨, 일반형은 일반 스위치 |
| 추천 공간 | 거실 / 방 / 주방 등 상품별 입력 |

주의:

- 확인되지 않은 인증번호, 사이즈, 제조원은 임의로 넣지 않는다.
- 가격은 넣지 않는다.
- 배송 안내는 넣지 않는다.
- 제조국 / MADE IN KOREA는 사용자가 삭제 요청한 상품에서는 넣지 않는다.

### 5-6. 설치 전 확인 영역

```text
설치 전 반드시 전원을 차단해 주세요.
천장 고정 상태와 배선 상태를 확인해 주세요.
제품 크기와 설치 공간을 확인한 뒤 설치해 주세요.
설치 후 점등 상태를 확인해 주세요.
```

### 5-7. 교환 안내 영역

교환 안내가 필요한 경우에만 별도 섹션으로 넣는다.

```text
교환 안내
- 제품 수령 후 제품 상태를 먼저 확인해 주세요.
- 제품 이상 또는 오배송이 확인될 경우 교환 접수가 가능합니다.
- 박스 개봉 후에는 단순 변심으로 인한 반품이 불가합니다.
- 설치 및 사용 후에는 제품 상태에 따라 교환이 제한될 수 있습니다.
- 교환 접수 시 제품 박스, 구성품, 포장 상태를 함께 확인해 주세요.
```
