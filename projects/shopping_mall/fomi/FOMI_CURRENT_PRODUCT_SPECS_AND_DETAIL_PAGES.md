# Fomi Current Product Specs and Detail Page Rules

최종 업데이트: 2026-06-30

이 문서는 포미 LED 조명 상품 상세페이지 작업 중 확인한 상품별 스펙, 상세페이지 구성, 이미지 제작 기준을 한곳에 모은 현재 기준 문서다.

## 1. 공통 상세페이지 제작 기준

```text
최종 상세페이지 가로폭: 500px
세로 길이: 제한 없음 / 내용 전체가 들어가도록 자동 확장
기준 표현: 500px × auto
```

- 이미지와 글자는 눌려 보이면 안 된다.
- 가로와 세로 비율을 강제로 찌그러뜨리지 않는다.
- 생성 또는 편집 결과가 724px, 627px 등으로 나오더라도 최종 판매용 파일은 500px 가로 기준으로 리사이즈한다.
- 세로는 내용 전체가 들어가도록 필요한 만큼 길게 둔다.
- 글자 겹침, 중복 글자, 기존 글자 잔상, 잘린 글자가 있으면 재작업한다.
- 최종 다운로드용 파일명은 영어만 사용하고 공백 대신 언더바 `_`를 사용한다.

## 2. 상품 이미지 정확도 기준

```text
상품 이미지는 반드시 동일한 실제 상품 이미지여야 한다.
```

- 등 이미지는 실제 판매 상품과 동일한 상품 이미지를 사용한다.
- 제품 외형을 임의로 새로 그리거나 다른 모델로 대체하지 않는다.
- 모양, 모서리, 두께, 프레임, 패널 수, 리모컨 유무를 바꾸지 않는다.
- 배경, 문구, 표, 섹션 디자인은 새로 구성할 수 있지만 상품 자체는 바꾸지 않는다.
- 60W, 120W, 150W, 180W처럼 모델이 다르면 서로 대체해서 쓰지 않는다.
- 설치 컷을 만들 때도 실제 상품 이미지 기준으로 같은 상품이 설치된 것처럼 구성한다.

## 3. 넣지 않는 내용

포미 상세페이지 공통 제외 기준:

```text
가격 제외
배송 안내 제외
원산지 제외
MADE IN KOREA 제외
원산지 스펙 행 제외
로고 제외
워터마크 제외
불필요한 FAQ 제외
```

기본형 주광색 상품에는 아래 기능을 넣지 않는다.

```text
삼색변환 제외
밝기조절 제외
리모컨 제외
전구색 / 주백색 / 주광색 3색 기능 설명 제외
```

삼색변환 리모컨형 상품에만 아래 표현을 사용할 수 있다.

```text
삼색변환
밝기조절
리모컨 제어
전구색 / 주백색 / 주광색
```

3. 짧은 소개 문구
4. 실제 공간 설치 컷

```text
주광색 6500K

```

## 6. 삼색변환 리모컨형 상세페이지 구성

삼색변환 리모컨형 상품은 아래 구성으로 만든다.

```text
1. 상단 타이틀
2. 실제 상품 이미지 히어로 섹션
3. 공간 분위기 조절 소개
4. 실제 공간 설치 컷
5. 핵심 기능: 삼색변환 / 밝기조절 / 리모컨 제어
6. 하나의 조명, 3가지 분위기 섹션
7. 디자인 섹션
8. 제품 스펙표
9. 설치 및 사용 전 확인
```

사용 가능 색상 표현:

```text
전구색: 따뜻한 분위기
주백색: 은은한 분위기
주광색: 깨끗한 분위기
```

## 7. 공간 설치 컷 기준

- 거실등: 밝고 모던한 거실 배경 사용
- 방등: 밝고 깔끔한 침실 / 방 배경 사용
- 주방등: 밝고 모던한 주방 배경 사용
- 조명 색상이 주광색이면 화이트, 쿨그레이, 밝은 모던 배경과 맞춘다.
- 설치 컷에서도 실제 상품과 패널 수, 형태, 비율을 맞춘다.
- 설치 컷은 상세페이지 중 소개 문구 아래 또는 핵심 기능 위에 배치한다.

## 9. 작업 시 주의사항

- 상품 페이지에서 제공된 스펙 이미지가 있으면 그 스펙을 우선한다.
- URL 본문이 직접 열리지 않을 때는 사용자가 올린 스펙 이미지 기준으로 확정한다.
- 상품명에 리모컨 / 삼색변환 / 밝기조절이 없으면 해당 기능을 넣지 않는다.
- 제품 외형이 긴 1패널, 2패널, 3패널인지 먼저 확인한다.
- 120W 기본형 거실등과 120W 삼색변환 리모컨형은 서로 다른 상품으로 구분한다.
- 60W 원형방등, 60W 사각방등, 60W 거실등, 60W 주방등은 서로 다른 상품으로 구분한다.
- 상세페이지가 생성형 이미지로 724px 등 다른 가로폭으로 나오면 최종 제출 전 반드시 500px로 조정한다.
- 글자 겹침이 있으면 확대/축소로 해결하지 말고 섹션 간격을 다시 잡는다.
