# Fomi Product Image and Detail Page Guide

이 문서는 포미 LED 조명 상품의 대표이미지 제작 방식과 상세페이지 구성 방식을 새 대화창에서도 같은 형식으로 이어가기 위한 작업 기준이다.

## 1. 기본 역할

쇼핑몰 상세페이지와 대표이미지를 제작할 때는 다음 기준을 우선한다.

- 사용자가 준 이미지, 상품명, 상품페이지 정보, 인터넷 확인 정보 중 필요한 내용만 사용한다.
- 이미지는 사용자가 “그려줘”, “만들어줘”라고 요청할 때만 생성한다.
- 가격은 대표이미지와 상세페이지 이미지에 넣지 않는다.
- 배송 안내는 넣지 않는다.
- 교환 안내는 사용자가 요청한 경우에만 별도 섹션으로 넣는다.
- 글자는 무조건 크게 키우는 것이 아니라, 잘 보이게 배치한다.
- 글자 뒤 배경 박스나 과한 장식은 최대한 피한다.
- 이미지의 밝기, 여백, 대비를 고려해서 글자가 선명하게 보이도록 한다.

## 2. 파일명 규칙

다운로드용 파일명은 항상 영어로 저장한다.

- 한글 파일명 금지
- 공백 대신 언더바 `_` 사용
- 간단한 영어 상품명 + 옵션 + 번호 형식 사용
- 대표이미지 번호는 `_01`, `_02`, `_03`, `_04`, `_05` 형식

예시:

```text
Fomi_LED_Living_Room_Light_150W_01.jpg
Fomi_LED_Living_Room_Light_120W_Remote_01.jpg
Fomi_LED_Room_Light_75W_Remote_01.jpg
Fomi_LED_Round_Room_Light_60W_Daylight_01.jpg
Fomi_LED_Square_Room_Light_60W_01.jpg
Fomi_LED_Kitchen_Light_60W_01.jpg
Fomi_LED_Kitchen_Light_25W_01.jpg
```

상세페이지 파일명 예시:

```text
Fomi_LED_Living_Room_Light_150W_Detail_Page.jpg
Fomi_LED_Kitchen_Light_60W_Detail_Page.jpg
```

```text

포미 LED 거실등 50w 주백색
```

대표적으로 작업 대상은 다음과 같다.

## 4. 대표이미지 제작 기준

대표이미지는 보통 상품당 5개를 만든다.

### 4-1. 공통 규칙

- 1:1 정사각형 이미지로 만든다.
- 쇼핑몰 대표이미지용으로 깔끔하고 눈에 잘 띄게 만든다.
- 고급스럽고 클릭률이 높아 보이게 만든다.
- 가격, 배송 문구, 교환 문구, 로고, 워터마크, 불필요한 글자는 넣지 않는다.
- 제품 외형은 사용자가 준 실제 이미지 기준으로 최대한 정확히 유지한다.
- 같은 상품이라도 5개 디자인은 서로 다르게 만든다.
- 상품 형태를 잘못 추정하지 않는다.
- 조명 모듈 개수, 패널 배열, 리모컨 유무를 반드시 확인한다.

### 4-2. 조명 색상과 배경 색감 맞춤

실내 배경이 들어갈 때는 조명 색상과 배경 색감을 맞춘다.

- 주광색 / 깨끗한 화이트 조명: 화이트, 쿨그레이, 밝은 모던 실내 배경
- 주백색 / 은은한 조명: 아이보리, 연그레이, 내추럴톤 배경
- 전구색 / 따뜻한 조명: 베이지, 우드톤, 따뜻한 실내 배경

주의:

- 화이트 조명에 노란 베이지 후광을 과하게 넣지 않는다.
- 배경과 등 색상이 따로 놀지 않게 전체 톤을 통일한다.
- 제품 윤곽이 약하면 배경을 쿨그레이 또는 다크그레이로 조정한다.

### 4-3. 추천 대표이미지 5가지 스타일

상품별로 아래 5가지 중에서 구성한다.

1. **쿨그레이 제품 단독 컷**
   - 밝은 회색 천장 또는 스튜디오 배경
   - 제품 윤곽이 잘 보이도록 그림자와 테두리 강조

2. **다크그레이 고대비 컷**
   - 진회색 또는 차콜그레이 배경
   - 흰색 LED등이 강하게 돋보이게 표현
   - 쿠팡/네이버 썸네일에서 눈에 잘 띄는 스타일

3. **실내 설치 컷**
   - 거실, 방, 주방 등 실제 사용 공간에 설치된 느낌
   - 배경은 제품 조명 색과 맞게 선택
   - 제품이 너무 작아지지 않도록 상단에서 명확히 보이게 배치

4. **프리미엄 카탈로그 컷**
   - 화이트 또는 라이트그레이 배경
   - 제품을 크게 확대
   - 여백과 그림자로 고급스럽게 표현

5. **라이프스타일 컷**
   - 공간 분위기를 보여주는 컷
   - 조명은 주인공으로 유지
   - 배경은 깔끔하고 과하지 않게 처리

## 5. 상품별 외형 고정 기준

대표이미지 생성 전 제품 외형을 반드시 고정한다.

### 포미 LED 거실등 120W 국산 친환경조명

- 긴 사각 라운드 등 2개
- 좌우로 나란히 배치
- 모서리가 둥근 직사각형
- 얇은 테두리 라인 있음
- 천장 밀착형
- 리모컨 없음
- 삼색변환 예시 없음
- 4개, 5개, 6개 패널로 만들면 안 됨

파일명 예시:

```text
Fomi_LED_Living_Room_Light_120W_01.jpg
```

### 포미 LED 거실등 60W 국산 친환경조명

- 긴 사각 라운드 LED등 1개
- 천장 밀착형
- 얇은 테두리 라인 있음
- 제품 단독 컷에서는 다크그레이 또는 쿨그레이 배경이 잘 맞음

파일명 예시:

```text
Fomi_LED_Living_Room_Light_60W_01.jpg
```

### 포미 LED 주방등 60W 국산 친환경조명

- 길고 슬림한 주방등 1개
- 라운드 처리된 긴 직사각형
- 주방 설치 컷 가능
- 화이트/쿨그레이 배경 또는 모던 주방 배경이 적합

파일명 예시:

```text
Fomi_LED_Kitchen_Light_60W_01.jpg
```

### 포미 LED 주방등 25W 국산 친환경조명

- 주방등 60W보다 짧고 슬림한 긴 직사각형
- 라운드 처리된 양 끝
- 단독 컷은 쿨그레이 또는 다크그레이가 적합
- 설치 컷은 주방 배경 사용 가능

파일명 예시:

```text
Fomi_LED_Kitchen_Light_25W_01.jpg
```

### 포미 LED 방등 75W 삼색변환 밝기조절 리모컨

- 사각 라운드 방등 1개
- 리모컨 포함 가능
- 삼색변환 상품이므로 색상 예시를 넣을 수 있으나, 사용자가 삭제 요청하면 일부 또는 전체 제거
- 리모컨은 실제 사진 참고

파일명 예시:

```text
Fomi_LED_Room_Light_75W_Remote_01.jpg
```

### 포미 LED 거실등 150W 삼색변환 밝기조절 리모컨

- 사각 라운드 거실등 1개
- 리모컨 포함 가능
- 삼색변환 / 밝기조절 표현 가능
- 가격 문구 금지
- 배송 문구 금지

파일명 예시:

```text
Fomi_LED_Living_Room_Light_150W_Remote_01.jpg
```

## 7. 상세페이지 구성 기준

상세페이지는 긴 세로형 이미지로 구성한다.

### 7-1. 기본 흐름

1. 상단 메인 비주얼
   - 제품명
   - 핵심 기능 2~3개
   - 제품 이미지

2. 제품 핵심 설명
   - 공간에 어울리는 조명
   - 깔끔한 디자인
   - 사용 장면 설명

3. 기능 설명
   - 삼색변환 여부
   - 밝기조절 여부
   - 리모컨 여부
   - 주광색 / 주백색 / 전구색 등 색상 정보

4. 디자인 설명
   - 라운드 모서리
   - 슬림한 테두리
   - 천장 밀착형
   - 심플한 실내 연출

5. 제품 스펙표
   - 필요한 정보만 정리
   - 가격, 배송 안내 제외

6. 설치 및 사용 전 확인
   - 전원 차단
   - 천장 상태 확인
   - 배선/고정 확인
   - 리모컨 상품이면 리모컨 작동 확인

7. 교환 안내
   - 사용자가 요청한 경우 별도 표기
   - 박스 개봉 후 단순 변심 반품 불가 문구 포함 가능

### 7-2. 상세페이지 글자 기준

- 글자는 크게만 하지 않는다.
- 이미지 위에 글자가 올라가면 대비와 여백을 확보한다.
- 글 뒤 배경 박스는 최대한 피한다.
- 너무 많은 문구를 넣지 않는다.
- 제품 구매 전환에 필요한 문구만 사용한다.
- 문장은 짧고 명확하게 쓴다.

### 7-3. 상세페이지 스펙표 예시

포미 LED 거실등 150W 리모컨형 기준 예시:

```text
제품명: 포미 LED 거실등
소비전력: 150W
정격: 220V / 60Hz / 150W
안전인증: HD11260-23032
제조원: ㈜에이아이조명
사이즈: W800 × D800 × H45mm
조명 방식: LED
색상 기능: 전구색 / 주백색 / 주광색
주요 기능: 삼색변환 / 밝기조절
조작 방식: 리모컨
설치 방식: 천장 설치형
추천 공간: 거실 / 방 / 넓은 실내 공간
```

주의:

- 제조국 / MADE IN KOREA는 사용자가 삭제 요청한 경우 넣지 않는다.
- 가격은 넣지 않는다.
- 배송 안내는 넣지 않는다.

### 7-4. 교환 안내 예시

```text
교환 안내
- 제품 수령 후 제품 상태를 먼저 확인해 주세요.
- 제품 이상 또는 오배송이 확인될 경우 교환 접수가 가능합니다.
- 박스 개봉 후에는 단순 변심으로 인한 반품이 불가합니다.
- 설치 및 사용 후에는 제품 상태에 따라 교환이 제한될 수 있습니다.
- 교환 접수 시 제품 박스, 구성품, 포장 상태를 함께 확인해 주세요.
```

## 8. 쿠팡 카테고리 기준

포미 LED 거실등, 방등, 주방등은 쿠팡 등록 시 보통 다음 카테고리를 우선 검토한다.

```text
홈인테리어 > 조명/스탠드 > 천장등/실링라이트
```

상품명 예시:

```text
포미 LED 거실등 150W 삼색변환 밝기조절 리모컨 천장등
포미 LED 주방등 60W 국산 친환경조명 천장등
```

## 9. 새 대화창에서 이어가는 방법

새 대화창에서는 다음처럼 말하면 된다.

```text
GitHub ai_brains 저장소의 projects/shopping_mall/fomi/FOMI_PRODUCT_IMAGE_AND_DETAIL_PAGE_GUIDE.md 기준으로 포미 상품 대표이미지 만들어줘.
```

또는:

```text
포미 작업 기준 문서 보고 같은 형식으로 상세페이지 만들어줘.
```

그러면 이 문서 기준으로 대표이미지와 상세페이지 구성을 이어가면 된다.
