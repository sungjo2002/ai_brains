# Fomi Detail Page Size Standard

이 문서는 포미 쇼핑몰 상세페이지 이미지 제작 시 사용할 가로폭과 비율 기준이다.

## 기본 기준

```text
판매용 최종본 기본 가로폭: 860px
세로 길이: 고정하지 않음
기준 표현: 860px × auto
```

## 작업 원칙

- 상세페이지는 가로폭을 860px로 맞춘다.
- 세로 길이는 무시하고, 내용이 전부 들어가도록 필요한 만큼 길게 만든다.
- 이미지와 글자는 눌려 보이면 안 된다.
- 가로와 세로 비율을 강제로 찌그러뜨리지 않는다.
- 크기 조절 시 원본 비율을 유지한다.
- 새 이미지를 추가할 때도 전체 가로폭은 860px로 통일한다.
- 섹션 사이 여백은 자연스럽게 유지하고, 빈공간이 과하게 생기지 않게 정리한다.
- 원본이 313px처럼 작은 경우에는 판매용 최종본으로 바로 쓰지 않고, 860px 기준으로 재제작하거나 비율 유지 리사이즈본과 구분한다.
- “선명도는 건들지 말라”는 요청이 있으면 샤픈, 보정, 재생성, 임의 확대 보정을 하지 않는다.

## 한 줄 기준

```text
포미 상세페이지 최종본은 860px × auto 기준으로 만들고, 세로는 내용 전체가 들어가도록 제한하지 않으며, 이미지와 글자가 눌려 보이지 않게 원본 비율을 유지한다.
```
