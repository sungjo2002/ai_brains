# Fomi Product Image Accuracy Standard

이 문서는 포미 쇼핑몰 상세페이지와 대표이미지를 만들 때 상품 이미지 정확도를 지키기 위한 기준이다.

## 핵심 원칙

```text
상품 이미지는 반드시 동일한 실제 상품 이미지여야 한다.
```

## 작업 기준

- 상세페이지에 들어가는 등 이미지는 실제 판매 상품과 동일한 상품 이미지를 사용한다.
- 대표이미지, 상세페이지, 스펙 설명 이미지 모두 실제 상품 외형을 기준으로 한다.
- 제품 외형을 임의로 새로 그리거나 생성하지 않는다.
- 제품 모양, 모서리, 두께, 프레임, 패널 수, 리모컨 유무를 바꾸지 않는다.
- 삼색변환 표현도 실제 상품 이미지 또는 사용자가 제공한 동일 상품 이미지 기준으로 구성한다.
- 배경, 여백, 문구, 표, 섹션 디자인은 새로 만들 수 있지만 상품 자체는 바꾸면 안 된다.
- 상품 이미지가 부족하면 임의 생성하지 않고, 사용자가 제공한 이미지나 상품페이지에서 확인한 동일 상품 이미지를 기준으로 작업한다.
- 동일한 포미 상품군이라도 120W, 150W, 180W처럼 모델이 다르면 서로 대체해서 쓰지 않는다.

## 상세페이지 재작업 기준

- 등 이미지 = 실제 상품 이미지 그대로 사용
- 리모컨 이미지 = 실제 상품 리모컨 이미지 기준 사용
- 배경/문구/스펙표/섹션 구성 = 새로 디자인 가능
- 제품 외형/모듈 수/크기감/테두리/광원 형태 = 임의 변경 금지

## 한 줄 기준

```text
포미 상세페이지와 대표이미지 작업에서는 상품 이미지를 새로 그리지 말고, 반드시 실제 판매 상품과 동일한 상품 이미지를 사용한다.
```
