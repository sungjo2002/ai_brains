# P002 아이디어

## 프로젝트명

1인 기업 AI 사무실

## 한 줄 정의

1인 기업 운영자가 혼자 처리하던 사무, 개발, 쇼핑몰, 문서, 고객응대, 이미지, 서버관리 업무를 로컬 AI와 자동화 도구로 보조하는 개인 업무 운영 시스템이다.

## 핵심 목표

- 24시간 기본 사무 자동화 흐름을 만든다.
- Ollama와 LM Studio를 로컬 AI 엔진으로 사용한다.
- Antigravity / VS Code / Cursor 안에서 AI 작업자를 운용한다.
- connect-ai 소스를 참고해 명령 입력, 파일 생성, 지식 정리, GitHub 저장 구조를 만든다.
- 지식 창고와 기억 저장소를 분리해 AI가 회사 기준과 작업 이력을 기억하게 한다.
- 나중에 Cafe24, ComfyUI, n8n, Gmail, GitHub, 서버관리 기능을 단계적으로 연결한다.

## 만들고 싶은 최종 모습

- 명령 입력기에서 자연어로 업무 지시
- 역할별 AI 직원 선택
- 지식 창고 검색 후 답변
- 기억 저장소에서 성조님 업무 기준 반영
- 작업 결과를 Markdown/DB/GitHub에 저장
- 승인 필요한 작업은 사용자 확인 후 실행
- 쇼핑몰 상품등록, 고객문의, 이미지 생성, 개발/서버관리까지 확장

## 핵심 화면

- 대시보드
- 명령 입력기
- AI 직원
- 지식 창고
- 기억 저장소
- 작업 기록
- 상품 등록
- 고객 문의
- 이미지 생성
- GitHub / 개발
- 설정

## 1차 범위

1차는 전체 자동화가 아니라 기반 시스템 구축이다.

- P002 기준 문서 작성
- connect-ai 분석
- Ollama / LM Studio 연결 확인
- 명령 입력기 설계
- 지식 창고 구조 설계
- 기억 저장소 구조 설계
- GitHub 저장 규칙 확정
