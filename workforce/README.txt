# workforce README

이 파일은 인력관리 프로그램의 현재 작업 상태를 기록한다.

## 최신 상태

- PC 최신 수정본: pc_65_table_column_user_resize_priority_fix.zip
- 서버 최신 수정본: server_10_attendance_clear_tombstone_fix.zip
- 모바일 최신 수정본: mobile_16_attendance_retry_queue_fix.zip
- 현재 PC 작업은 PC 화면 안정화와 사용성 정리 중심으로 진행 중이다.
- 깜빡임 문제는 일부 개선됐지만 완성된 홈 화면의 잔깜빡임은 보류 상태다.
- 다음 작업은 홈화면/근로자 목록 표 컬럼 조절 방식 재정리다.

## 완료

- 배너 관련 문제 일부 개선 완료.
  - 홈 배너 빈 배지 박스 제거.
  - 배너 늦게 표시되는 느낌 개선.
  - 페이지별 배너 정상 표시 확인.
- 왼쪽 메뉴 클릭 시 흰 화면 깜빡임은 상당 부분 개선 완료.
- 로그인 후 홈 화면이 뜨기 전 빈 화면 깜빡임은 개선 완료.
- 로그인 화면 디자인 정리 완료.
  - 2번 시안 방향 적용.
  - 기존 프로그램 로고 사용.
  - 제목을 관리자 로그인으로 변경.
  - 설명 문구 삭제.
  - 전체 크기 축소.
  - 로고 크기 재조정.
- 급여관리 화면 보강 완료.
  - 우측 이중 스크롤 제거 보강.
  - 급여표 로딩 속도 개선.
  - 급여관리 전체 화면이 외부 스크롤처럼 움직이는 문제 보강.
- 홈화면 인원 목록에서 표시번호를 맨 앞으로 이동하고 제목을 번호로 변경 완료.
- 근로자 목록에서 이름, 영문이름 칸 폭이 너무 좁은 문제 1차 보강 완료.
- 홈화면/근로자 목록 컬럼 폭 조절 및 저장 기능을 시도했으나 아직 정상 완료 아님.

## 진행 중

- 홈화면과 근로자 목록의 표 컬럼 조절 방식 정리 중.
- 현재 pc_65 기준 문제:
  - 홈화면에서 하단 가로 스크롤이 생김.
  - 홈화면의 현재 상태 컬럼 크기 조절이 안 됨.
  - 화면 전체는 고정되어야 하고, 컬럼 경계선만 움직여야 한다.
  - 사용자가 컬럼 하나를 넓히면 다른 컬럼이 줄어드는 방식이어야 한다.
  - 표 전체 폭은 고정되어야 한다.
  - 하단 가로 스크롤은 생기면 안 된다.
  - 마지막 현재 상태 컬럼도 조절 가능해야 한다.

## 보류

- 완성된 홈 화면에서 보이는 잔깜빡임은 다음으로 보류.
- 시작/종료 속도 개선은 레이아웃/표 사용성 정리 후 진행.
- 모바일 고도화, 서버 동기화 고도화, OCR 고도화는 PC 1차 정리 후 진행.

## 오류/주의

- pc_64는 표 컬럼 자동 보정과 사용자 조절이 충돌하여 실패로 판단한다.
- pc_65도 아직 최종 성공이 아니다. 홈화면 하단 스크롤과 현재 상태 컬럼 조절 불가 문제가 남아 있다.
- 표 컬럼 조절 수정 시 자동 폭 보정이 사용자의 마우스 조절 중 끼어들면 안 된다.
- 표 전체 폭은 화면 안에 고정해야 하며, 사용자가 조절하는 컬럼 경계만 움직여야 한다.
- 경계선은 보이게 하되 기존 표 색보다 진하면 안 된다.
- 기능 조건, 저장 조건, 동기화 조건, 계산 조건, 권한 조건은 건드리지 않는다.
- 저장소 기록보다 실제 업로드 파일과 사용자가 확인한 결과를 우선한다.
