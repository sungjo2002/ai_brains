## 오류/주의

- 민감정보는 저장하지 않는다.
- 서버 비밀번호, 토큰, 개인식별번호, 비밀키는 저장 금지.
- 저장소는 public 상태이므로 DB, 백업, 실제 설정 파일, 운영 키를 올리지 않는다.
- `pc_38` 이후 PC/서버 동기화는 서버 `.env`의 `PC_SYNC_KEY`와 PC 설정의 `pc_sync_key`가 같아야 정상 작동한다.
- `mobile_14` 기준으로 차량 상태는 정상 확인됐지만, app.js 안의 서비스워커 등록 버전이 남아 있을 수 있으므로 다음 모바일 수정 시 버전 일치 여부 확인 필요.
- 저장소 작업기록이 실제 파일 상태와 다르면 현재 업로드 파일과 사용자가 확인한 실행 결과를 우선한다.
