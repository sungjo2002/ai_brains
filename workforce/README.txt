이 파일은 구형 구조에서 사용하던 작업 상태 파일이다.

현재 기준에서는 사용하지 않는다.

## 새 기준 위치

- 최신 상태: `projects/P001/CURRENT.md`
- 다음 작업: `projects/P001/TASKS.md`
- 작업 기록: `projects/P001/HISTORY.md`
- 규칙: `projects/P001/RULES.md`
- 명령어: `projects/P001/COMMANDS.md`
