# PROJECTS

이 저장소의 작업은 프로젝트 ID 기준으로 관리한다.

## P001

- 이름: 인력관리 프로그램
- 상태: 진행 중
- 위치: `projects/P001`
- 최신 상태: `projects/P001/CURRENT.md`
- 다음 작업: `projects/P001/TASKS.md`
- 고정 규칙: `projects/P001/RULES.md`
- 실행/검사: `projects/P001/COMMANDS.md`
- 테스트 기준: `projects/P001/TESTS.md`

## 다음 프로젝트 등록 규칙

- 새 프로젝트는 `P002`, `P003`처럼 번호를 이어서 만든다.
- 상위 폴더명에는 프로젝트 이름을 직접 쓰지 않는다.
- 프로젝트 이름과 설명은 각 프로젝트의 `META.md`에 기록한다.
